
Module that provide meta information that can be used by Twitter when links to the
site are contained within tweets. You can share teaser, date, author information to
create previews of your sites content on Twitters website or mobile app's

Twitter dev reference to Twitter cards can be found here;
https://dev.twitter.com/docs/cards

Installation
------------

Download and copy the module to your sites modules directory then enable it.
Set what content types this should work for at admin/config/services/twitter-cards
